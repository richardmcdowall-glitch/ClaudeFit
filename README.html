Getting Started

1
Clear old data (if updating)
⚙️ Settings → Profile → Clear All Data
2
Set up your profile
Same page — fill in your name, weight target, calorie range, W/kg goals, FTP, and Z2 weekly target. Tap Save.
3
Get a Claude API key
Go to console.anthropic.com/settings/keys
Create a key, paste it into the API Key field in Profile. This powers AI food estimation.
4
EatingWithClaude (optional)
Double-tap the 🍽️ icon → opens the standalone app. Paste the same API key in its Settings.

App Guide

🏠 Home 🏋️ Gym 🍽️ Food ⏱️ Progress ⚙️ Settings
Swipe from the left edge for a date sidebar. Swipe from the right edge for a quick calendar strip.
🏠 Home

Your morning check-in. Enter today's weight and resting heart rate, set mood and readiness (1–5 sliders).

The Z2 field logs zone 2 minutes for commutes and untracked rides.

Below: weight/target/W/kg cards, consistency ring, and body map showing muscles trained in the last 10 days. Tap any muscle for detail.

🏋️ Gym

A scrolling day list. Tap any day to expand and log your session.

  • ⭕ marks a set done → starts rest timer
  • ⇊ copies set 1 to all sets
  • Feel indicators per exercise (tap to cycle)
  • 💬 note per exercise
  • 🧘 Mobility logs a standard mobility session
  • 📅 Move relocates a session to another date
  • Plans in the header jumps to plan management

The meso banner shows your periodisation phase. Tap to change phase or cycle number.

🍽️ Food

Notes tab: Type what you ate, tap Process for AI macro estimation, or 📷 for photo estimation. Needs API key.

Quick Add: Fast entry — name, calories, protein.

Favourites: Tap ★ on any meal in History to save. One-tap to log again.

Import: Paste MEAL lines from chat. Preview before saving.

Double-tap the 🍽️ nav icon to open EatingWithClaude for more food options.

⏱️ Progress

Charts and trends. Use 7/14/30/∞ to zoom.

  • Body weight with target line and 7-day average
  • Calories colour-coded against your floor/ceiling
  • W/kg weekly averages with FTP and goal lines
  • EF trend (watts ÷ HR) for Z2 and spinning
  • Zone 2 weekly totals vs your target
⚙️ Settings

Profile: All your targets, FTP, API key, Clear All Data

Plans: View, edit, clone, reactivate training plans

+ New Plan: Build from scratch with autocomplete

Exercises: Auto-built registry, muscle group overrides

Export: Gym, food, plan, or full JSON dumps

Manual: In-app help for every feature

💾 Backup & Restore

Tap 💾 in the header to create a JSON backup. Copy and save it somewhere safe.

To restore: tap Restore in the header → load a file or paste JSON.

Covers everything: health, gym, plans, food, exercise registry, profile, and preferences.

💡 Tips
  • The rest timer floats in the corner after marking a set done. Tap to dismiss.
  • EF below 1.0 flags fatigue 🚨 — useful for spotting overtraining.
  • Z2 ride data flows from Z2 CTRL via copy-paste on each day.
  • All data stays in your browser. Nothing is sent to any server except AI food calls (direct to Anthropic using your key).
